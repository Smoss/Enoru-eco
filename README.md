# Installation instructions

Requirements: 
- Node.js
- Python 2.7 or 3.3

1. Download the project
2. run `npm install` followed by `bower install`
3. Run the server:

  For Python 2.7:
    `python -m SimpleHTTPServer`

  For Python 3.x:
    `python3 -m http.server`
4. Go to `localhost:8000/game/`
